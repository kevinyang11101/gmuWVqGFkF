# 酒店管理系统 java491

一款基于 Vue3 和 Node.js 技术栈开发的酒店管理系统，包括酒店后台管理系统和酒店预订前台系统。

## 技术栈

- 前端：Vue3
- 后端：Node.js

## 功能模块

### 后台管理系统

- 酒店信息管理
- 房间类型管理
- 房间状态管理
- 订单管理
- 客户信息管理

### 酒店预订前台系统

- 房间查询
- 预订房间
- 订单支付
- 个人信息管理

## 系统角色

- 管理员：负责后台管理系统的操作
- 用户：使用前台系统进行房间预订和支付

## 运行环境

- 前端：现代浏览器
- 后端：Node.js 环境

## 部署步骤

1. 准备 Node.js 运行环境
2. 安装项目依赖
3. 运行后端服务
4. 前端项目编译并运行
